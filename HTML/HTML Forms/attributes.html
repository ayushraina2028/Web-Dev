Input Attributes

Here we are going to see different types of attributes that are used in different input elements.

1. Placeholder

This is like a temporary display value. It will help user what kind of value this value input field is expecting from us. When we write some value this value will be vanished

2. Value

This is like a initial value that we provide to input field, it acts in different ways depending on what input element we are using. We can say this as default value of a input element in html. When we delete this default value then we will see placeholder data in input field

4. Name

When we will submit the form, how the receiver will know about particular input fields, For example, there are 2 input fields of type text and one is for name and other is for address and user may put correct details in incorrect one also. For receiver to decide which data belongs to which input field we use name attribute.

Here are some examples
1. If we give name = 'firstname' attribute in input tag, then the receiver will know that for user entered value is belongs to input field containing name as firstname.

4. min, max, maxlength attribute

These are used to put the constraints on the input fields so that we get required data without any non necessary data.

Here is example discussed below
Suppose there is a form with number input field with some constraints that value must be in range [10,100]
Suppose there is form with text input field with constraints that length of text must be less than specific number(10)
Now if our inputs satisfy the above constraints then only we can submit the form otherwise a popup will appear.

5. autofocus and size attribute

When we click any input field, a blue border appears on it, but only when we click it. If we give autofocus in any input field then by default blue border will be there on that input field. Its usecase can be that it shows us starting of the form.
Size attribute will help in determining the width of input field
